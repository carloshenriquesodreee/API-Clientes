# API-Clientes


Primeiro utilize o comando:
-npm install

Segundo configure o arquivo 
.env

Depois utilize o comando:
npx sequelize db:create

Depois utilize o comando:
npx sequelize db:migrate

Por ultimo o comando :
npm run debug
